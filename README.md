# COMP3005Assignment3Q1
Code and documentation for COMP3005Assignment3Q1

--Set Up Instrutions--
1. In PostgreSQL create database called Students
2. Use the database creation code in newly created database query tool (in PostgreSQL) to create a students table
3. Populate table using same query tool (in PostgreSQL) using database population code (provided in assignment instructions but also included here)
4. Ensure postgresql-42.7.8.jar is included in external libraries of IdeaProject Assignment3Q1
5. Inside Main.java file located in IdeaProject Assignment3Q1 folder src, ensure both the localhost port is correct and the PostgreSQL password are correct
6. Run Main.java file
